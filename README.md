# File Management App

This is a simple File Management App written in Python. The application allows users to create, view, delete, read, and edit files within the current directory.

## Features

* **Create File** : Create a new file with a specified name.
* **View All Files** : List all files in the current directory.
* **Delete File** : Delete a specified file.
* **Read File** : Read the content of a specified file.
* **Edit File** : Append data to a specified file.
* **Exit** : Exit the application.

## Prerequisites

* Python 3.x

## Code Overview

### `create_file(filename)`

Creates a new file with the given filename. If the file already exists, it prints an error message.

### `view_all_files()`

Lists all files in the current directory. If no files are found, it prints an appropriate message.

### `delete_file(filename)`

Deletes the file with the given filename. If the file does not exist, it prints an error message.

### `read_file(filename)`

Reads and prints the content of the file with the given filename. If the file does not exist, it prints an error message.

### `edit_file(filename)`

Appends user input to the file with the given filename. If the file does not exist, it prints an error message.

### `main()`

Runs the main loop of the application, presenting the user with a menu of options to manage files.
